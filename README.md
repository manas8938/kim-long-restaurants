## üìñ Description

Kim Long Restaurants is a modern, responsive restaurant website I developed for a client. The project showcases a beautiful Chinese restaurant with an interactive menu, customer reviews, contact information, and smooth animations. Built with the latest React 19 and TypeScript for type safety, styled with Tailwind CSS for a sleek design, and deployed on Netlify for fast performance.

## ‚ú® Features

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **Interactive Menu** - Browse dishes by categories with beautiful images
- **Customer Reviews** - Showcase customer testimonials and ratings
- **Contact Form** - Easy-to-use contact form with validation
- **Location Map** - Google Maps integration for directions
- **Smooth Animations** - Engaging scroll and hover animations
- **Fast Performance** - Optimized with Vite for lightning-fast loading
- **SEO Friendly** - Proper meta tags and semantic HTML

## üöÄ Quick Start

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation
git clone https://github.com/manas8938/kim-long-restaurants.git
cd kim-long-restaurants
npm install
npm run dev

### Build for Production

```bash
npm run build
npm run preview
```
